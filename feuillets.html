Feuillets paroissiaux

Les feuillets paroissiaux sont disponibles sous format PDF ou encore HTML (page web classique). Dans le cas des fichiers PDF, la mise en page est scrupuleusement fidèle au feuillet qui serait imprimé. Dans l’autre cas, il y aura manifestement un peu plus de fantaisie, mais les informations seront néanmoins les mêmes.

Lorsqu’il y a conflit entre les versions imprimées et électroniques, il faut savoir que les versions électroniques (pages web et fichiers PDF ci-dessous) affichent parfois des modifications de dernière minute qui n’ont pu être apportées à temps sur la version imprimée.
